# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2026-01-13

### Added
- Interactive copy button to dictionary popup for easier text copying
- Delay to resume video playback when hovering over dictionary popup to prevent interruption

### Changed
- Updated copyright years to 2025-2026 in all relevant files
- Changed ruby background to transparent for improved visibility
- Improved hover behavior for dictionary popup

### Updated
- README and help documentation to include option for no tone coloring in subtitle settings

## [1.0.1] - 2026-01-03

### Added
- Option for no tone coloring in subtitle settings
- Dark background theme support
- Margin to icons for better spacing

### Changed
- Updated release action to v2

## [1.0.0] - 2025-12-30

### Added
- Initial release of Youku Dual Subtitle Chrome Extension
- Core functionality for dual subtitle display
- Dictionary popup feature
- Chinese character tone coloring
- Subtitle settings and customization options
- Release workflow automation
